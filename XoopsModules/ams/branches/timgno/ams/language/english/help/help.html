Help: ams

Description

ams is a module to creates a Slashdot-like article section, where users can post articles and comment.

Install/uninstall

No special measures necessary, follow the standard installation process - extract the /ams folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.
